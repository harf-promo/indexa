# How-to guides

Task-focused recipes. For the conceptual overview start with the
[Quickstart](../quickstart.md); for every config option see the
[Configuration Reference](../config.md).

- **[Index a repo and export it for Claude / Cursor](index-and-export-for-claude.md)** — turn a
  codebase into one portable context file an AI tool can read.
- **[Serve your index to AI agents over MCP](live-retrieval-over-mcp.md)** — let Claude Desktop,
  Cursor, or any MCP client browse and search your local index live.
- **[Debug the code graph](debug-the-code-graph.md)** — `graph` / `who_calls` / `blast_radius`,
  and how to read them honestly (bare-name matching caveat).
- **[Tune Indexa for a small machine](tune-for-a-small-machine.md)** — keep an 8–16 GB laptop
  responsive while it builds context.

When something misbehaves, see [Troubleshooting](../TROUBLESHOOTING.md) (doctor-first triage).
