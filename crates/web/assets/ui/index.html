Welcome to Indexa — your local context engine

Point Indexa at a folder and it builds a searchable context map of everything inside — entirely on your machine. After that, your AI tools answer from it instead of re-reading your files every time.

  1. Add a folder. A code repo, your Documents, a notes vault — anything.
  2. Indexa builds context. It scans, embeds, and summarizes locally with Ollama. No files leave your machine.
  3. Ask or export. Query in plain language, or export a context file to Claude, Cursor, or any AI tool.
or run indexa scan <path> in your terminal

Advanced Features

Audio transcription
whisper binary

Requires whisper-cli (whisper.cpp) installed separately. Applies on the next indexa deep run.

Video captions
Vision model

Requires ffmpeg (frame extraction) + an Ollama vision model. Samples 1 frame/2s, max 8 frames per video. Not yet counted by memory watchdog — enable only with headroom.

Boost or suppress files, directories, or classification categories in search results. Weight 2.0 = double score; 0.1 = almost hidden; 1.0 = neutral.
